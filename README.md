# Comercial Mariano — Sistema de Processos

App estático servido via NGINX no Railway.

## Como publicar

1. Faça upload desta pasta para um repositório no GitHub
2. No Railway: **New Project → Deploy from GitHub repo**
3. Selecione o repositório
4. Railway detecta o Dockerfile automaticamente e faz o deploy
5. Clique em **Generate Domain** para obter a URL pública
